## Maven Repository and Dependency
```xml
<repository>
    <id>inventoryApi</id>
    <url>https://raw.github.com/PolygonDev/InventoryApi/repository/</url>
</repository>

<dependency>
    <groupId>de.PolygonDev</groupId>
    <artifactId>inventoryapi</artifactId>
    <version>1.1.8</version>
    <scope>provided</scope>
</dependency>
```
